# 📅 Calendar Suite 

Коллекция из семи реализаций **календаря с управлением событиями** на разных языках программирования.

## Общие возможности

- 📆 Просмотр календаря на месяц (с возможностью переключения вперёд/назад)
- ➕ Добавление событий на конкретную дату (название, описание, время)
- ✏️ Редактирование и удаление событий
- 💾 Автоматическое сохранение событий (локальный JSON / localStorage)
- 🔍 Поиск событий по названию
- 📋 Список событий на выбранный день
- 🌙 Тёмная/светлая тема (в GUI и веб-версиях)

## Языки и технологии

| Язык      | Интерфейс          | Хранение                |
|-----------|--------------------|-------------------------|
| Python    | Tkinter            | JSON файл               |
| JavaScript| Веб (HTML/CSS/JS)  | localStorage + экспорт  |
| Go        | Веб (встроенный)   | JSON файл               |
| Rust      | CLI + цветной вывод| JSON файл               |
| Java      | Swing              | JSON (Jackson)          |
| C#        | Windows Forms      | JSON (Newtonsoft.Json)  |
| PHP       | Веб (серверный)    | Файл или сессия         |
